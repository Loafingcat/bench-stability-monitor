# Bench Stability Monitor

## 분석 영상

https://studio.youtube.com/channel/UCzbKLeTUWfJjLpU4kOoAMVw

## 프로젝트 소개

Bench Stability Monitor는 고중량 벤치프레스 수행 중 발생할 수 있는 부상 위험을 영상 분석을 통해 탐지하는 프로젝트입니다.

실제 1RM(One Rep Max) 측정 영상에서 부상을 경험했던 사례를 바탕으로, 운동 중 나타나는 비정상적인 움직임을 정량적으로 분석하고 위험도를 평가하는 것을 목표로 개발했습니다.

---

## 문제 정의

고중량 벤치프레스(영상의 경우 130kg)에서는 아주 작은 궤적 변화나 바벨의 비틀림만으로도 어깨, 흉근, 회전근개 등의 부상 위험이 급격하게 증가할 수 있습니다.

하지만 이러한 위험 신호는 대부분 육안으로 확인하기 어렵고, 운동 후 영상을 다시 보더라도 정확한 판단이 쉽지 않습니다.

따라서 영상만으로 운동 중 발생하는 이상 움직임을 자동으로 탐지하고 위험도를 수치화하는 시스템을 구현하고자 했습니다.

---

## 시행착오

### 1. 바벨 직접 검출 시도

처음에는 YOLO 객체 탐지 모델을 이용해 바벨과 원판을 직접 검출하고 기울기를 분석하는 방식을 고려했습니다.

하지만 일반 객체 탐지 모델은 바벨을 안정적으로 인식하지 못했고, 학습용 데이터셋도 충분하지 않았습니다.

또한 촬영 환경과 조명에 따라 검출 결과가 크게 달라져 신뢰성이 낮았습니다.

---

### 2. 손목 기울기 기반 분석

다음으로는 YOLOv8 Pose를 사용하여 양손 손목 위치를 추적한 뒤, 손목 사이의 각도를 계산하여 바벨 기울기를 측정하는 방식을 시도했습니다.

그러나 실제 촬영 영상은 정면이 아닌 대각선 각도였기 때문에 문제가 발생했습니다.

손목 간 수평 거리가 작아지는 순간 각도가 비정상적으로 크게 계산되었고, 정상 동작에서도 80도 이상의 기울기가 발생하는 등 오탐이 많았습니다.

---

### 3. 카메라 각도 문제 발견

분석 과정에서 촬영 각도가 결과에 큰 영향을 준다는 사실을 확인했습니다.

정상 자세에서도 왼손과 오른손의 높이가 완전히 동일하지 않았으며, 이를 그대로 분석에 사용하면 위험 신호가 과도하게 발생했습니다.

이를 해결하기 위해 운동 초반 안정 구간을 기준(Baseline)으로 설정하고 이후 움직임을 상대적인 변화량으로 계산하도록 수정했습니다.

---

## 최종 접근 방식

여러 실험 끝에 가장 신뢰도가 높았던 지표는 **오른손 손목의 수평 궤적 변화**였습니다.

실제 부상이 발생한 영상을 분석해보니 부상 직전부터 손목이 정상 경로를 벗어나기 시작했고, 특정 시점 이후 급격한 측면 이동이 발생했습니다.

이를 기반으로 다음과 같은 분석 파이프라인을 구성했습니다.

### 분석 과정

1. YOLOv8 Pose로 프레임별 관절 좌표 추출
2. 오른손 손목 위치 추적
3. 운동 초반 안정 구간에서 기준 위치(Baseline) 계산
4. 프레임별 수평 이탈 거리 측정
5. 위험 임계치를 초과하는 구간 탐지
6. 지속 시간과 이탈 정도를 기반으로 위험 점수 계산

---

## 위험도 산정 요소

### 1. 수평 궤적 이탈

가장 중요한 지표입니다.

정상적인 벤치프레스에서는 손목이 일정한 경로를 유지해야 하지만, 부상 직전에는 갑작스러운 측면 이동이 발생했습니다.

이를 통해 비틀림(Twist) 발생 여부를 판단합니다.

### 2. 위험 구간 지속 시간

일시적인 흔들림보다 위험 상태가 얼마나 오래 지속되는지를 함께 평가합니다.

### 3. Sticking Point

가슴 근처에서 바벨이 정체되는 구간을 탐지합니다.

실패 직전 상황에서는 이 구간이 길어지는 경향을 보였습니다.

### 4. 관절 검출 신뢰도 변화

한쪽 손목이 급격하게 흔들리거나 프레임에서 사라지는 경우 Pose 모델의 Confidence가 감소하는 현상이 나타났습니다.

이를 추가 위험 신호로 활용했습니다.

## 시각화 기능

분석 결과는 오버레이 영상으로 생성됩니다.

영상에는 다음 정보가 표시됩니다.

- 실시간 위험 상태(STABLE / WARNING / DANGER)
- 손목 추적 궤적
- 기준선(Baseline)
- 위험도 점수
- 위험 이벤트 발생 시점
- Twist 구간 시각화

이를 통해 단순 수치뿐 아니라 실제 어떤 움직임이 위험 판정의 원인이 되었는지 직관적으로 확인할 수 있습니다.

---

## 분석 결과

실제 부상이 발생했던 1RM 벤치프레스 영상 분석 결과

- 최대 수평 이탈: 83px 이상
- 위험 구간 발생
- Sticking Point 확인
- 한쪽 손목 검출 신뢰도 급감
- 최종 위험도: 90점 이상

특히 위험 구간이 실제 부상 발생 시점과 거의 일치하는 결과를 확인할 수 있었습니다.

## 프로젝트 의의

이 프로젝트는 단순 자세 교정이 아닌 실제 부상 위험 신호를 탐지하는 데 초점을 맞췄습니다.

특히 데이터셋이 거의 존재하지 않는 문제를 해결하기 위해 여러 접근 방식을 실험했고, 최종적으로 실제 부상 사례에서 나타난 손목 궤적 변화를 핵심 지표로 활용하여 위험도를 분석할 수 있는 시스템을 구현했습니다.
